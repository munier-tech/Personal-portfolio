# Quick Start Guide

## Installation & Setup (2 minutes)

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Start development server:**

   ```bash
   npm run dev
   ```

3. **Open in browser:**
   - Go to http://localhost:3000

## First Steps (5-10 minutes)

### Step 1: Update Your Name

- Open `app/components/Navbar.tsx`
- Change `&lt;Muniir /&gt;` to your name

### Step 2: Add Your Bio

- Open `app/components/Hero.tsx`
- Update the headline and description

### Step 3: Add Your Projects

- Open `app/components/Projects.tsx`
- Replace projects in the `projects` array with your work

### Step 4: Add Contact Info

- Open `app/components/Contact.tsx`
- Update email, phone, and location

### Step 5: Add Social Links

- Open `app/components/Navbar.tsx` and `app/components/Footer.tsx`
- Update social media links

## Testing

- **Dark Mode**: Click sun/moon icon in navbar
- **Responsive**: Resize browser or test on mobile
- **Contact Form**: Fill out form and check validation

## Build for Production

```bash
npm run build
npm start
```

## Deploy

### Option 1: Vercel (Recommended)

1. Push to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import project
4. One-click deploy!

### Option 2: Netlify

1. Push to GitHub
2. Connect to Netlify
3. Deploy

### Option 3: Self-hosted

```bash
npm run build
npm start
```

Then deploy the `.next` folder to your server.
